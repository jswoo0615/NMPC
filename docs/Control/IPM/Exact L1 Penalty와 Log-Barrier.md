이 문서는 NMPC의 Line Search 과정에서 `merit` 스칼라 함수가 어떻게 추종 성능, 제어 effort, 그리고 물리적 제약 조건 (장애물 회피 등)의 균형을 맞추어, 특히 물리적 제약을 절대적으로 강제 (Enforce)하는지 설명하는 명세서입니다.

#### 1. `L1_WEIGHT`의 정의와 역할 (Exact Penalty)
##### 1.1. 흔한 오해 : "위반했을 때만 벌점을 준다?"
일반적으로 제약 조건 $c(x) \le 0$ (예 : 안전거리 침범)을 다룰 때, 값이 양수 (위반)가 될 때만 패널티를 준다고 생각하기 쉽습니다. 하지만 Merit 함수에 사용된 L1 패널티는 완전히 다르게 동작합니다.
##### 1.2. 엄밀한 일관성 강제 (Enforcing Consistency)
IPM 솔버는 부등식 제약을 풀기 위해 가상의 양수 슬랙 변수 $s \ge 0$를 도입하여 억지로 등식 (Equality)으로 만듭니다.
$$c(x) + s = 0$$
Merit 함수에 포함된 L1 패널티 항은 다음과 같습니다.
$$w_{L1} |c + s + \alpha ds|$$
이 절댓값 $(|\cdot|)$ 항은 제약 조건 위반 여부 상관없이, "물리적 현실 $(c)$과 솔버의 뇌 속에 있는 가상 변수 $(s)$의 아귀가 완벽하게 들어맞는가?"를 검증합니다.
- 차가 장애물에서 5m 떨어져 있어 매우 안전하더라도 $(c = -5)$, 솔버 내부의 슬랙 변수가 $s = 3$으로 계산되었다면 $|-5 + 3| = |-2| = 2$가 되어 $2000 (w_{L1} \times 2)$이라는 엄청난 벌점을 부여합니다.
- 즉, 이 항은 **물리적 상태와 수학적 상태의 단 1mm 오차도 허용하지 않는 엄밀성 유지 장치**입니다.

#### 2. 두 메커니즘의 완벽한 분업 (Log-Barrier vs L1 Penalty)
이 시스템은 두 가지 극단적인 메커니즘을 결합하여 절대적인 강건성을 확보합니다.

##### A. Log-Barrier (장기적 방향성 / 소프트 제약)
$$-\mu \ log(s + \alpha ds)$$
- **작동 방식** : 내부 슬랙 변수 $s$가 0에 가까워지는 것 자체에 무한대 $(\infty)$의 벌칙을 줍니다.
- **목적** : "제약 조건 영역 안에 머물러야 한다"라는 장기적인 목표를 설정합니다. 궤적이 경계선에 닿지 않고 항상 안전한 내부 (Interior, $s > 0$)에 머물도록 부드럽지만 단호하게 밀어냅니다.
##### B. L1 Penalty (즉각적 통제 / 하드 제약)
$$w_{L1} \ |c + s + \alpha ds| \quad (w_{L1} = 1000.0)$$
- **작동 방식** : 물리적 현실 $(c)$과 내부 변수 $(s)$의 합이 0에서 벗어나는 순간 즉각적으로 거대한 선형 벌칙을 가합니다.
- **목적** : Line Search 과정에서 솔버가 성능 (비용 함수 감소)에 눈이 멀어 제약 조건을 무시하는 스텝 $(\alpha)$을 밟으려 할 때, 무자비한 가중치로 `merit` 값을 폭발시켜 **해당 스텝을 즉각 기각하는 물리적 브레이크** 역할을 합니다.

#### 3. 왜 "Exact (엄밀한)" 가중치인가?
일반적인 제곱 패널티 $(c^{2})$를 사용하면 아무리 가중치를 키워도 최적화 솔버는 비용 함수를 줄이기 위해 제약 조건을 '미세하게 침범'하는 타협 (Trade-Off)을 해버립니다. (Inexact Penalty)

하지만 L1 패널티 $(|c|)$를 사용하고, 그 가중치 $(w_{L1})$를 시스템의 쌍대 변수 최댓값 $(\lambda_{max})$보다 충분히 크게 ($1000.0$ 등) 설정하면, 수학적으로 "비용 함수를 아무리 희생하더라도 제약 조건은 단 1mm도 ㅊ미범하지 않는 엄밀한 최적해 (Exact Optimal Solution)"가 도출됨이 증명되어 있습니다.
